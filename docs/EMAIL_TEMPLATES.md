## Template Variables Reference

| Variable | Description |
|----------|-------------|
| `{customerName}` | Customer's name |
| `{quoteNumber}` | Quote reference number |
| `{workOrderNumber}` | Work order number |
| `{invoiceNumber}` | Invoice number |
| `{projectDescription}` | Brief project description |
| `{quoteTotal}` | Total quote amount |
| `{orderTotal}` | Total order amount |
| `{amountDue}` | Amount due on invoice |
| `{depositAmount}` | Required deposit amount |
| `{estimatedCompletionDate}` | Expected completion date |
| `{senderName}` | Employee name |
| `{senderPhone}` | Employee phone |
| `{senderEmail}` | Employee email |
| `{companyPhone}` | Company main phone |
| `{companyAddress}` | Company address |
| `{portalLink}` | Customer portal URL |
| `{approvalLink}` | Proof/quote approval URL |
| `{paymentLink}` | Online payment URL |
| `{trackingNumber}` | Shipping tracking number |
| `{trackingLink}` | Carrier tracking URL |
| `{carrier}` | Shipping carrier name |
| `{currentStation}` | Current production station |
| `{currentStatus}` | Current order status |

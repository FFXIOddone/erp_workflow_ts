# File Management Instructions

## The New Way: Use the App

**DO NOT manually create folders or save files directly to the network drive.**

All files for work orders should be uploaded through the ERP app. The system will automatically:
- Create the correct folder structure
- Name files consistently
- Organize by file type (Proofs, Artwork, Emails, Print Files, etc.)
- Make files easy to find later

---

## How to Add Files to an Order

1. Open the work order in the app
2. Click the **"Network Files"** section
3. Click **"Upload File"**
4. Select the file type (Proof, Artwork, Email, Print File, etc.)
5. Choose your file
6. Done! The file is now saved and organized automatically.

---

## Why This Matters

- **Finding files is faster** - everything is in a predictable location
- **No more duplicate folders** - one folder per order, period
- **Proofs stay with orders** - no more hunting through email folders
- **Backups work better** - consistent structure = reliable backups

---

## Questions?

If you need to find an old file that isn't in the app yet, check the network drive manually. For all new work, use the app.

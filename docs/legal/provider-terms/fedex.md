# FedEx Developer Terms Summary

Official sources:

- `https://developer.fedex.com/api/legal/en-us/eula.pdf`
- `https://developer.fedex.com/api/en-us/sign-EULA.html`

Summary:

- FedEx API use is governed by the FedEx developer EULA and related registration steps.
- The person accepting the agreement must have authority to bind the company or employer.
- FedEx credentials, tokens, and API access should be kept private and used only for the authorized
  business purpose.
- Tracking, shipping, and related API responses remain subject to FedEx's current terms and limits.

# Microsoft API Terms Summary

Official source:

- `https://learn.microsoft.com/en-us/legal/microsoft-apis/terms-of-use`

Summary:

- Microsoft API access is controlled by Microsoft API terms of use.
- Applications should be registered and credentials should be kept secret.
- Microsoft grants a limited, revocable license subject to the API terms.
- Only the minimum data needed for the scenario should be collected and used.
- Unauthorized scraping, copying, reverse engineering, or bypassing of limits is not allowed.
